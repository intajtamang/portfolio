    • SKILLS

    • HTML

      HTML, which stands for Hyper Text Markup Language, is the standard markup language for creating Web pages. It describes the structure of a Web page using a series of elements. These elements label pieces of content, such as headings, paragraphs, links, and more, instructing the browser on how to display them.

    • CSS

      CSS, or Cascading Style Sheets, defines how HTML elements are displayed across different media. It streamlines design by controlling the layout of multiple web pages simultaneously. External stylesheets, stored in CSS files, enhance organization and reusability.

    • JAVASCRIPT

      JavaScript is a lightweight, cross-platform, single-threaded, and interpreted compiled programming language. It is also known as the scripting language for webpages. It is well-known for the development of web pages, and many non-browser environments also use it.

    • PYTHON

      Python is a versatile and beginner-friendly programming language known for its simplicity and readability, suitable for various applications from web development to data science.

    • DJANGO

      Django is a high-level Python web framework designed for rapid development, offering built-in features like ORM, authentication, and admin interface for efficient web application creation.

    • MYSQL

      MySQL is an open-source relational database management system (RDBMS) known for its reliability, scalability, and ease of use, widely used for managing structured data in web applications.
